### Manual Setup

Clone the repository

```bash
git clone https://github.com/appsecco/dvna
cd dvna
```

Install Dependencies

```bash
npm install
```

Start the application

```bash
npm start
```
Access the application at [http://localhost:9090](http://localhost:9090)
